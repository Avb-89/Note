Если у вас проблема с подключением к гиту который стоит в контейнере:

1) проверьте что вы прокинули 22 порт в хост (например 2221:22)
2) для подключения по ssh используйте альтернативный порт
 url = ssh://git@192.168.0.16:2221/AVB/vagrant.git

если клонируете то git clone ssh://git@192.168.0.16:2221/AVB/vagrant.git
