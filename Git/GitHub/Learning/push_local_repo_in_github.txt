Git - Заливаем гит репозиторий на гитхаб

Подготовка:

    У вас должна быть папка с проэктом на локальном компьютере, и созданый репозиторий на githab.com

Реализация:

    Указываем репозиторий:

    # git remote add origin https://github.com/ваш аккаунт/название репозитория.git  (посмотреть команды можно прям при создании репы)

    Проверяем:

    # git remote -v

    Заливает содержимое папки в гитхаб
    
    # git push -u origin "название ветки"
