Git - обновляем данные в репозитории на гитхаб

Проверяем, не изменилось ли что нибудь в репе:

    # git pull

Если изменилось:

    # git push

Формируем список файлов на отправку:

    # git add .

Добавляем коментарии для быстрого поиска (описываем что мы сделали):

    # git commit -m 'add git howto'

Синхранизируем с сервером

    # git push
