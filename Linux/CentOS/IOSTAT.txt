Команда iostat сообщает об использовании ЦП и статистику ввода/вывода дисков.
iostat собирает данные из файловой системы ргос, выдавая по одной строке для каждого физического устройства.

для установки используем:
# yum install sysstat

Использование:

-N - выводить информативные имена томов, соответствующие выводу утилиты df -h
-dx - вывод в расширенном виде, можно указать для отдельного диска, например iostat -dx /dev/dm-11
-t - добавляет время запроса в вывод

--------------------------------------------------------------------------------

Чтобы увидеть нагрузку в реальном времени:

# watch -d -n 1 "iostat -dx"

--------------------------------------------------------------------------------

Чтобы собрать статистику по загрузке диска можно исопльзовать

iostat -xk -t 10

(iostat -xk -t 10 | awk '// {print strftime("%Y-%m-%d %H:%M:%S"),$0}' >> iostat.log &)

Будут видны скачки в определнный промжуток времени.
