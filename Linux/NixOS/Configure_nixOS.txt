
После установки, если вы поменяли конфиг и хотите его применить, используйте:
	# nixos-rebuild switch 

Если вы решили что то попробовать, но не хотите сохранять изменения используйте (изменения исчезнут после перезагрузки, но конфиг будет с последними изменениями)

	# nixos-rebuild test

Работа с пакетами:

Установка пакетов:

	$ nix-env -i firefox python vim
	
Удаление:

	$ nix-env -e python

Обновление реп:

	$ nix-channel --update
	
Обновление пакетов:

	$ nix-env -u
	
Информация о пакете:

	$ nix-env -qaP | grep vim

Описание пакета:

	$ nix-env -qaP --description | grep vim
  
  
хранит все в одной директории /nix, не захламляет систему, удаляется одной командой rm -rf /nix
репа на гитхабе, это очень удобно (https://github.com/NixOS/nixpkgs)
