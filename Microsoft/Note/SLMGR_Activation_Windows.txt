slmgr Ключики
slmgr /

Для Windows
Cscript slmgr.vbs /upk - удалить ключ
Cscript slmgr.vbs /ipk XXXXX-XXXXX-XXXXX-XXXXX-XXXXX - установить ключ

Для Office
PS C:\Program Files (x86)\Microsoft Office\Office16> cscript OSPP.VBS /dstatus   

"Ipk" = Установить ключ продукта (заменяет существующий ключ)
"Upk" = Удалить ключ продукта
"Ato" = Активация Windows
"dli" = Показать информацию о лицензии (по умолчанию: текущая лицензия)
"Dlv" = Отображение подробных сведений о лицензии (по умолчанию: текущая лицензия)
"Xpr" = Дата истечения срока действия текущего состояния лицензии
"Cpky" = Очистить ключ продукта из реестра (предотвращает атаки, направленные на раскрытие информации)
"ILC" Установить лицензии
"Rilc" = Установите системные файлы лицензии
"Dti" = Показать Installation ID для оффлайн-активации
"ATP" = Активировать продукт с использованием кода подтверждения предоставленной пользователем
"Rearm" = Сброс лицензии состояния компьютера
"Rearm-app" = Сброс состояния лицензирования приложений
"Rearm-sku" = Сброс состояния лицензирования единицы хранения
"Skms" = Установить имя и / или KMS порт на вашем компьютере, вы будете использовать этот компьютер IPv6-
адрес должен быть указан в формате [имя]:. Порт""
"Ckms" = Очистить имя KMS компьютер, используемый (порт установлен в значение по умолчанию)
"Skms-domain" = Установка определенного домена DNS, в котором вы можете найти все KMS SRV записей. Эта
настройка не работает, если вы укажете одним службы управления ключами, устанавливается с помощью / SKMS
"Ckms-domain" = Очистка конкретного домена DNS, в котором вы можете найти все KMS SRV записей.
Характеризуется службы управления ключами, будет использоваться, если она была установлена ??с помощью
параметра / SKMS. В противном случае, он будет использовать по умолчанию KMS автоматического обнаружения
"Skhc" = Включить кэширование узла KMS
"Ckhc" = Выключите кэширования хоста KMS
"Sai" = Установить интервал (в минутах), чтобы использовать при попытке подключиться к услуге по
неактивированными клиентов KMS. Интервал активации должна быть в диапазоне от 15 минут (минимум) до 30
дней (максимальное значение), хотя рекомендуется, чтобы значение по умолчанию (2 часа)
"Sri" = Установить интервал обновления (в минутах), чтобы использовать при попытке подключиться к услуге
активированными клиентов KMS. Интервал обновления должен быть в диапазоне от 15 минут (минимум) до 30
дней (максимальное значение), хотя это и рекомендуется, что значение по умолчанию (7 дней)
"Sprt" = Настройка TCP порт, используемый службой KMS общаться с клиентами
"Sdns" = Включить DNS публикации на службу KMS (по умолчанию)
"CDNs" = Отключить DNS публикации на KMS сервис
"Spri" = Установка KMS приоритет Normal (по умолчанию)
"CPRI" = Установка KMS приоритет низкий
"Act-type" = Установка активации на 1 (AD), 2 (для KMS), 3 (маркер) или 0 (для всех)
"Lil" = Отображает список установленных лицензий публикации на основе маркеров активации
"Ril" = Удаляет выдачи лицензий, установленных на основе маркеров активации
"Ltc" = Отображает список сертификатов на основе маркеров активации
"Fta" = Force-активация на основе маркеров
"Ad-activation-online" = Активировать AD (Active Directory), используя ключ продукта, введенного пользователем
"Ad-activation-get-iid" = Показать Installation ID для леса AD (Active Directory)
"Ad-activation-apply-cid" = Включение AD (Active Directory), используя ключ продукта, введенного пользователем,
код подтверждения
"Ao-list" = Просмотр объектов активированных в AD (Active Directory)
"Del-ao" = Удаление объектов активации в AD (Active Directory) для объекта активации, предоставленные
пользователем
Командную строку запускаем с правами Администратора!
