

NS (name server) – это те записи, которые указывают на сервер имен для конкретного домена, важнейшая функция NS это делегирование домена.
MX (mail exchanger) — тип DNS-записи, предназначенный для маршрутизации электронной почты с использованием протокола SMTP. 
A это запись домена на сервере имен провайдера, которая указывает на IP-адрес
