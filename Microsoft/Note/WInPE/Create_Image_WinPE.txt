Создайте загрузочный DVD-диск, компакт-диск ISO-файл или виртуальный жестки диск со средой Windows PE (WinPE).
По умолчанию установка выполняется из памяти (также называется электронным диском), что позволяет извлечь USB-накопитель во время работы среды предустановки Windows.
Установка Windows ADK
Установите следующие компоненты из комплекта средств для развертывания и оценки Windows (ADK):
Средства развертывания: содержат среду средств развертывания и работы с образами.
Среда предустановки Windows: включает файлы для установки среды предустановки Windows.
Установка Windows PE на DVD, CD или файл ISO
Нажмите кнопку Пуск и введите развертывание. Щелкните правой кнопкой мыши элемент Среда средств развертывания и работы с образами и выберите команду Запуск от имени администратора. 
Создайте рабочую копию файлов Windows PE. Укажите либо x86, либо amd64:


copype amd64 C:\WinPE_amd64
Создайте файл ISO, содержащий файлы среды предустановки Windows PE:


MakeWinPEMedia /ISO C:\WinPE_amd64 C:\WinPE_amd64\WinPE_amd64.iso
Для записи на DVD-диск или компакт-диск: в проводнике щелкните правой кнопкой мыши файл ISO и выберите Записать образ диска > Запись на диск и следуйте указаниям.
При использовании Hyper-V
При работе среды предустановки Windows в Hyper-V рассмотрите возможность использования формата файла ISO вместо VHD для обеспечения быстрой установки виртуальной машины. Подробности см. в предыдущем разделе.
Установка среды предустановки Windows на виртуальный жесткий диск
Создайте виртуальный жесткий диск (.vhd или .vhdx):


diskpart
create vdisk file="C:\WinPE.vhdx" maximum=1000
attach vdisk
create partition primary
assign letter=V
format fs=ntfs quick
exit
Подготовьте диск при помощи MakeWinPEMedia:


MakeWinPEMedia /UFD C:\WinPE_amd64 V:
Отключите диск:


diskpart
select vdisk file="C:\WinPE.vhdx"
detach vdisk
exit
