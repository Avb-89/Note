Скриппт задает что должно быть установленно на сервере после отработки скрипта. (аналог доекркомпоса)

Configuration (это ключевое слово в начале скрипта DSC) installDHCP (Имя может быть любым)
{
    Node localhost (Указываем имена машин на которых надо это применить)
    {
        WindowsFeature (название командаплета) DHCPServer (название может быть любым) { 
            Ensure = "Present" (бывают 2 типа "Present" "Absent" - указывает добавляется или удаляется компонент или роль ) 
			*** ЧУВСТВИТЕЛЬНЫЙ К РЕГИСТРУ ***
            Name = "DHCP" (название роли которую надо поставить)
        }
    }
}

installDHCP (название конфигурации)

Start-DscConfiguration -Wait -Force -Path .\installDHCP -Verbose (запускает скрипт)

После исполлнения скрипта создает *.mof файл, который перезаписывается с каждым исполнением скрипта. Содержит информацию что происходло на сервере.
